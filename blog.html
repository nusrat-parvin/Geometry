What is DOM and what is the purpose of DOM?

Web documents include a programming interface called the Document Object Model (DOM).

What is the purpose of DOM?

Purpose of the DOM is to represenet the page so that program can change document's structure, appearance, and content. Programming languages can communicate with a page by interacting with the nodes and objects that the DOM uses to describe the document.It is a representation of the web page that is object-oriented and can be changed using a scripting language like JavaScript.

What is event bubble?

The Document Object Model (DOM) has a concept known as Event Bubbling. It occurs when an element receives an event, and that event propagates up the DOM tree to the element's parent and ancestor elements until it reaches the root element.
